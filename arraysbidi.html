Arrays: Vetor Bidimensional (Matriz)

Pode ser entendido como uma tabela com linhas e colunas. Ideal para organizar informações em formato de grade, como um tabuleiro de jogo.

Atividade 1 Atividade 2 Atividade 3 Atividade 4

Código da Atividade 1

Este programa lê uma matriz 3x3 de números inteiros fornecidos pelo usuário, em seguida solicita um valor para multiplicação. O programa multiplica cada elemento da matriz por esse valor e exibe o conteúdo da matriz resultante.


Código da Atividade 2

EEste programa lê uma matriz 2x2 de números inteiros fornecidos pelo usuário. Em seguida, solicita ao usuário que escolha uma linha da matriz e calcula a soma dos elementos dessa linha, exibindo o resultado.


Código da Atividade 3

Este programa cria uma matriz 2x2 em que cada elemento é calculado como a soma dos índices de sua posição na matriz. Em seguida, exibe o conteúdo completo da matriz.


Código da Atividade 4

Este programa cria uma matriz 2x3, solicita ao usuário que informe os valores de cada elemento, armazena esses valores na matriz e, em seguida, exibe todo o conteúdo da matriz.
